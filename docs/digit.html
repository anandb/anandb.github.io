Diagram Actions

  • Use "New Node" button to add nodes, or "Add Bounding Box" for grouping
  • Load SVG images using the image button for custom graphics
  • Drag nodes/elements to reposition, drag blue circles to resize
  • Right-click nodes to rename, change shapes, customize colors & add notes
  • Right-click tendrils to rename them and set visibility (exposed/internal)
  • Double-click nodes to enter inner diagrams for hierarchical modeling
  • Click orange tendrils to start edges, green tendrils to complete them
  • Drag tendrils to reposition them along element borders (left, right, top, bottom)
  • Ctrl+click elements to quickly create edges (auto-creates tendrils)
  • Edge labels appear in gaps within the edge line (not overlaid)
  • Arrow heads are unfilled for cleaner appearance
  • Right-click to cancel edge creation, right-click items to select for deletion
  • Press Delete key to remove selected items
  • Use Save/Load buttons to persist diagrams as JSON files
  • Export diagrams to SVG format for external use
